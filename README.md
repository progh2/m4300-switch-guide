# M4300-24X24F 자동할당(DHCP) 서버 구축 가이드

뉴미디어IT쇼(졸업작품 전시회)에서 사용할 **NETGEAR M4300-24X24F** 스위치를
DHCP 자동할당 서버로 구축하기 위한 **중학생도 따라 할 수 있는** 설정 가이드입니다.

## 📄 가이드 보기

- **웹페이지(배포본):** [GitHub Pages 링크](https://progh2.github.io/m4300-dhcp-guide/)
- 로컬에서 보기: `index.html` 파일을 브라우저로 열기

## 구성

- **네트워크 구조:** 스위치가 직접 게이트웨이(L3 라우팅) + DHCP 서버
- **외부망:** 고정 IP (업링크 포트)
- **내부 전시망:** `192.168.50.0/24`, DHCP 분배 `192.168.50.100 ~ .200`

## 주요 기능

- 🔧 **값 입력칸** — 외부망 IP 등 내 값을 한 번 입력하면 모든 명령어에 자동 반영
- 📋 **복사 버튼** — 명령어를 한 번에 복사
- 🖱️/⌨️ **웹(GUI) / 명령어(CLI) 탭** — 원하는 방식 선택
- 🆘 문제 해결 FAQ + 📖 용어 사전 포함

## 진행 단계

0. 준비물 → 1. 콘솔 접속 → 2. 공장 초기화 → 3. 비밀번호·관리IP
→ 4. 외부망 고정IP → 5. ⭐ DHCP 서버 → 6. 저장 & 테스트

## ⚠️ 중요 (인터넷 연결)

M4300은 NAT를 지원하지 않습니다. "스위치=게이트웨이" 구조에서 실제 인터넷이 되려면
외부망 담당자가 내부 대역(`192.168.50.0/24`)으로의 **경로(라우팅) 등록** 또는
외부 **NAT 장비**가 필요합니다. 자세한 내용은 가이드의 "네트워크 구조" 주의 박스를 참고하세요.
