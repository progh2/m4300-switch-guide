# M4300-24X24F 스위치(허브) 연결 가이드

뉴미디어IT쇼(졸업작품 전시회)에서 사용할 **NETGEAR M4300-24X24F** 스위치를
**여러 기기를 연결하는 스위치(허브)** 로 사용하기 위한 **중학생도 따라 할 수 있는** 가이드입니다.

> 처음에는 스위치를 DHCP 자동할당 서버로 만들려 했으나, M4300이 **NAT를 지원하지 않아**
> 인터넷 공유가 어려워 **단순 연결(L2 스위치)** 구조로 전환했습니다.
> IP 자동할당(DHCP)·인터넷 연결(NAT)은 **외부망 공유기/라우터(또는 학교망)** 가 담당합니다.

## 구성

- **네트워크 구조:** `[인터넷] → [공유기(고정IP·DHCP·NAT)] → [M4300 스위치(연결만)] → [전시 기기]`
- **스위치 역할:** 단순 연결(L2 스위치). 설정은 거의 없음 — 공장 초기화 후 케이블만 연결
- **(선택) 관리용 IP:** 나중에 웹으로 포트 상태 점검이 필요할 때만 설정

- 🖱️/⌨️ **웹(GUI) / 명령어(CLI) 탭** — (관리 IP 설정 시) 원하는 방식 선택
- 🔌 **케이블 연결 순서 + 루프 방지 경고**

## 진행 단계

0. 준비물 → 1. 콘솔 접속 → 2. ⭐ 공장 초기화 → 3. (선택) 관리용 IP
→ 4. 케이블 연결 → 5. 테스트

## ⚠️ 중요 (IP는 누가 나눠주나)

스위치는 IP를 나눠주지 않습니다. 전시 기기가 IP를 자동으로 받으려면
**외부망(공유기·라우터·학교망)에 DHCP가 켜져 있어야** 합니다.
외부망이 고정 IP 하나만 준다면 그 앞에 **인터넷 공유기 1대**를 두어
DHCP·NAT를 맡기세요(시중 가정용 공유기로 충분).
